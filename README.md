# Блог
Данный проект представляет собой веб-страницу со статьями(а также комментариями к ним) и страницами статей. Данные взяты с https://gorest.co.in/ 

## Как запустить проект
Склонируйте репозиторий на свой компьютер:

### `git clone` https://github.com/fluplix3/tableStudents.git
Откройте файл index.html в любом современном браузере.
